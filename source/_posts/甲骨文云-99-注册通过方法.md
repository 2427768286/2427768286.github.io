# 骚操作开始

当提示 {% span red, 处理交易时出错 %} 后，会发现无法继续 {% span red, 勾选 %} 同意协议 {% span red, 单选框 %} ，也就无法继续点击 {% span red, 开始我的免费试用 %} 。如果绕过这个限制呢？
在当前页面按下 {% span red, F12 %} ，进入开发者工具，如下图所示操作：

<!-- wp:image {"sizeSlug":"large"} -->
<figure class="wp-block-image size-large"><img src="https://cdn.jsdelivr.net/gh/2427768286/STDM-imgs/images/%E7%94%B2%E9%AA%A8%E6%96%87%E8%B6%8A%E8%BF%87%E9%93%B6%E8%A1%8C%E5%8D%A1%E6%97%A0%E6%B3%95%E9%80%9A%E8%BF%87%E8%AF%95%E7%94%A8%E9%AA%8C%E8%AF%81%E7%9A%84%E6%95%99%E7%A8%8B-08.jpg" alt=""/></figure>
<!-- /wp:image -->

代码：
```
$("#main > div > div.col-md-8.col-xs-12.mainContainer > div.contentContainer > div.agreementComponent > label > input[type=checkbox]").removeAttr("disabled");
```
此时会发现可以继续 {% span red, 勾选 %} 同意协议 {% span red, 单选框 %} 了。

接下来进入骚操作，首先这里不能再使用代理了，安卓手机在热点WIFI中提供了USB共享网络功能，使用数据线连接电脑和手机，手机开启USB共享网络，此时电脑就会走手机网络。手机关闭wifi，使用流量来操作。

循环执行下面的步骤：
<ol><li>手机开关流量（更换ip）</li><li>电脑 {% span red, 勾选 %} 同意协议 {% span red, 单选框 %} ，点击 {% span red, 开始我的免费试用 %} 。如果注册通过就停止循环，继续提示失败，则继续循环。</li></ol>

通常来说更换IP十几次就能注册成功了。脸白一次就成功也不是没有。
笔者通过这种方法，基本10分钟内就能注册一个账号。

# 注册验证规则猜想

接下来我来猜想一下甲骨文注册的验证规则。

* 通过注册多个账号的经验，可以知道其实除了信用卡信息、邮箱信息有用，其他信息都是无意义的。
* 通过不断更换IP，就能注册成功，说明甲骨文可能是对IP段做了限制。
* 虽说是通过不断更换IP注册成功的，但是不要忘了前面做的骚操作，取消了同意协议 {% span red, 单选框%} 的限制，提高了注册请求频率，所以也可能是我们抢的快了，所以就通过了。
